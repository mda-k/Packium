# PACKIUM
Packium is my pretty simple primitive project, that i hope one day will possibly be able to give people an easier and stupid-proof way to use winget, the windows package manager. (of course, you need to have it installed in order for this program to work...)
you can get winget here:
https://github.com/microsoft/winget-cli/

the main idea is that winget is a really good thing, but i suppose not a lot of people want to learn how to use a package manager, and most people are scared of the command prompt. so, i want to build something where you can use winget easier, and update any app, or every app you like, just with a click of a button and some waiting.
other plans include the ability to download stuff with winget, uninstall stuff with winget.
this is not professional at all, so don't have high expectations, i'm a rookie.
